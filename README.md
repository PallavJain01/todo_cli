Todo_CLI

Todo-app in the command line

```bash
Usage: cargo run -- [OPTIONS] [COMMAND]

Commands:
  add
  finish
  delete
  options
  help     Print this message or the help of the given subcommand(s)

Options:
  -l, --list [<PAGE>]  Todo List
  -h, --help           Print help
  -V, --version        Print version
```
